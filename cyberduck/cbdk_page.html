"A LOGO for game studio-Cyberduck"

"What is Cyberduck?"

It is a logo create by myself, based on the cyberpunk art style and the shape of game console. I want to be a game designer and set up my own game studio in the future. This LOGO was created in 2021.

"Main Features"

  • The hair of the duck represent the wire on game console
  • The eyes of the cyberduck represent the buttons on game console
  • The font I use for the LOGO is close to the font style in some video games
